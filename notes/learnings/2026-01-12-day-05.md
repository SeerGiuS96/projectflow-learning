## Key mental models
- Application defines contracts (ports); Infrastructure implements them.
- If two interfaces have the same name but different namespaces, DI will break in subtle ways.
- Ports belong in a dedicated folder to avoid accidental duplication.
- Program.cs should not become a dumping ground for service registrations.
